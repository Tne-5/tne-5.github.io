# tne-5.github.io

Project archive website for Tne-5.

The site is plain HTML/CSS so it can be published directly with GitHub Pages without a build step. Its structure is project-first: the homepage is an index, and each project page is intended to become a case study with process notes, visuals, evaluation, and links to a cleaned public repository.

## Publish

1. Create a public repository named `tne-5.github.io`.
2. Push this folder to the repository's `main` branch.
3. Enable GitHub Pages from `Settings > Pages`, using branch `main` and folder `/root`.
