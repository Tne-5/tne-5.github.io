# tne-5.github.io

Personal portfolio website for Tiago Fernandez-Nespral.

The site is plain HTML/CSS so it can be published directly with GitHub Pages without a build step. It links to the cleaned public project repositories in this portfolio kit.

## Publish

1. Create a public repository named `tne-5.github.io`.
2. Push this folder to the repository's `main` branch.
3. Enable GitHub Pages from `Settings > Pages`, using branch `main` and folder `/root`.
