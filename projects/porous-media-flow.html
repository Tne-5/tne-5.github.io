Porous Media Flow U-Net

A machine learning model for predicting flow fields through porous cross-sections, using CNN and U-Net architectures with physical scaling and symmetry-aware averaging.

Problem

Given binary cross-section masks and physical parameters, predict the flow field through a porous medium. The challenge is structured prediction: the output is a full 2D field, not a single label.

Approach

The cleaned implementation normalizes labels by a physical scale, trains U-Net variants, masks closed pixels, and averages predictions over horizontal/vertical/180-degree symmetries to respect the geometry of the task.

mask + parameters -> physical scaling q0
                  -> dimensionless target q*
                  -> U-Net / custom U-Net
                  -> symmetry averaged prediction
                  -> CSV submission output

Highlights

  • Structured image-to-image regression.
  • Dimensionless target normalization.
  • Custom boundary-channel U-Net.
  • Symmetry-aware ensemble wrapper.
