Porous Media Flow U-Net

A structured image-to-image regression project for predicting flow fields through porous cross-sections.

Problem

Given binary cross-section masks and physical parameters, predict the full 2D flow field through a porous medium.

Approach

The cleaned implementation normalizes labels by a physical scale, trains U-Net variants, masks closed pixels, and averages predictions over horizontal, vertical, and 180-degree symmetries.

mask + parameters -> physical scaling q0
                  -> dimensionless target q*
                  -> U-Net prediction
                  -> symmetry-averaged output

Highlights

  • Structured image-to-image regression
  • Dimensionless target normalization
  • Custom boundary-channel U-Net
  • Symmetry-aware ensembling
