This project predicts flow fields through porous cross-sections. Unlike a normal classification task, the output is a full 2D field, so the model has to preserve spatial structure from input to output.

Problem

The input is a porous geometry and physical parameters. The target is the corresponding flow field. Closed pixels should remain physically meaningful, and the model should generalize across different obstacle arrangements.

Modeling approach

Design choices

ChoiceWhy it matters
U-NetPreserves high-resolution spatial output while using deeper context.
Dimensionless targetNormalizes labels by a physical scale before learning.
Closed-pixel maskPrevents predictions from treating solid regions like open flow.
Symmetry averagingUses horizontal, vertical, and 180-degree transformations to stabilize predictions.

Output

The cleaned repository keeps the model and inference code used to generate field predictions and submission-style CSV outputs. It excludes datasets, local training artifacts, and generated model checkpoints.
